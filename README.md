
# 🚀 AI-Powered Startup Progress Monitoring Dashboard

An end-to-end workflow automation project built using **n8n**, **Google Forms**, **Google Sheets**, **AI**, and **Google Looker Studio** to automate startup progress tracking.

## 📌 Problem Statement

Collect weekly startup updates and automatically:
- 📈 Track KPIs
- 🤖 Generate AI-powered reports
- ⚠️ Highlight delayed milestones

## ✨ Features

- 📝 Automated weekly startup update collection using Google Forms.
- 📊 KPI tracking and Startup Health Score calculation.
- 🤖 AI-generated startup progress reports.
- ⚠️ Automatic delayed milestone detection.
- 📧 Automated email notifications.
- 📈 Interactive dashboard built with Google Looker Studio.

## 🛠️ Tech Stack

- Google Forms
- Google Sheets
- n8n Workflow Automation
- JavaScript
- Google Gemini / AI Integration
- Gmail
- Google Looker Studio

## 🔄 Workflow Architecture

```text
Google Form
      ↓
Google Sheets (Responses)
      ↓
n8n Workflow Trigger
      ↓
JavaScript KPI Calculation
      ↓
Milestone Status Detection
      ↓
AI Report Generation
      ↓
Google Sheets (Reports)
      ↓
Gmail Notification
      ↓
Google Looker Studio Dashboard
```
